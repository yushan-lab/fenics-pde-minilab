# Repository Rules

- Do not fabricate numerical results.
- Every figure and table referenced by `README.md` must be generated by scripts in this repository.
- Keep code small, readable, and reproducible.
- Demonstrate a traditional finite element workflow: geometry, mesh, boundary conditions, weak form, solver, post-processing, error analysis, and visualization.
- Use `Makefile` commands for reproduction.
- If DOLFINx cannot run in the current environment, document the limitation clearly and leave numerical results marked as pending.
- `docs/cv_bullets.md` must contain only honest resume bullets supported by actual repository content.
